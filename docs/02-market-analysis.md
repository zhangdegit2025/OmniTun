# OmniTun — 市场分析与用户研究

## 一、市场规模

### 1.1 总体可寻址市场（TAM）

全球内网穿透 / 网络隧道服务市场：

| 维度 | 2024 年 | 2027 年（预测） | CAGR |
|------|---------|-----------------|------|
| 全球 Zero Trust 网络访问（ZTNA）市场 | $12.3B | $26.2B | 28.4% |
| 开发者工具 SaaS 市场 | $15.6B | $32.1B | 27.2% |
| IoT 连接管理平台市场 | $5.8B | $14.7B | 36.3% |
| 远程访问工具市场 | $4.2B | $8.9B | 20.6% |
| **TAM（交叉去重后）** | **$18.0B** | **$38.0B** | **26.4%** |

OmniTun 的 TAM 取 ZTNA + 开发者工具 + 远程访问的交叉部分。

### 1.2 可服务市场（SAM）

聚焦 **开发者和中小企业的隧道/暴露/组网需求**：

| 细分 | 2024 年 SAM | 2027 年 SAM | 渗透率天花板 |
|------|------------|------------|-------------|
| 开发者个人隧道暴露 | $0.8B | $2.2B | 35% |
| 中小企业组网 | $1.5B | $3.8B | 25% |
| 企业内部穿透 | $2.1B | $4.5B | 20% |
| IoT 设备穿透 | $0.6B | $1.8B | 15% |
| **SAM 合计** | **$5.0B** | **$12.3B** | — |

### 1.3 可获得市场（SOM）

**Year 1 目标 SOM**：$50M

- 假设 Year 1 获取 5,000 付费用户
- ARPU $250/月（Team 级以上占比 30%）
- 占 SAM 的 **1.0%**

**Year 3 目标 SOM**：$250M

- 目标 25,000 付费用户
- ARPU $830/月（Enterprise 占比 20%）
- 占 SAM 的 **1.7%**

## 三、用户旅程与场景地图

### 3.1 核心场景矩阵

| 场景 | 频率 | 协议 | 拓扑 | 关键需求 |
|------|------|------|------|----------|
| 本地 Webhook 调试 | 每天多次 | HTTP/HTTPS | 中继 | 1秒内获取 URL、自动 TLS、请求重放 |
| 远程同事预览 | 每天 | HTTP/HTTPS | 中继 | 稳定的 URL、Basic Auth、带宽不限 |
| CI/CD 回呼通知 | 持续 | HTTP/WebSocket | 中继 | WebSocket 直通、连接保持 |
| 数据库远程连接 | 偶尔 | TCP | P2P/中继 | 安全隧道、IP 白名单、端口转发 |
| K8s 集群暴露 | 长期 | TCP/HTTP/gRPC | Mesh | Ingress Controller、自动服务发现 |
| IoT 设备 OTA | 按需 | TCP/UDP | P2P | UDP 支持、大规模设备管理、离线缓冲 |
| 分布式团队组网 | 持续 | TCP/UDP | Mesh | 自动打洞、加密传输、零配置 |
| 客户演示 | 偶尔 | HTTPS | 中继 | 自定义域名、密码保护、流量统计 |

### 3.2 典型用户 Day-1 旅程

```
发现 OmniTun
    │
    ├─> GitHub README → 10 秒安装 (`brew install omnitun` / `curl | sh`)
    ├─> 注册账号 (`omnitun login`)
    │
    ├─> 第一个隧道 (`omnitun http 8080`)
    │   → 立即获取 https://xxx.omnitun.io
    │   → 自动签发 TLS 证书
    │   → Web Dashboard 查看实时流量
    │
    ├─> 配置自定义域名 (`omnitun tunnel create --domain api.mydomain.com`)
    │   → DNS CNAME 配置引导
    │   → 自动签发 Let's Encrypt 证书
    │
    ├─> 设置访问控制 (`omnitun tunnel update --auth basic --user admin:pass`)
    │
    ├─> 邀请团队成员 (`omnitun workspace invite colleague@company.com`)
    │
    └─> 升级 Pro/Team（按需自然升级）
```

## 五、定价心理学研究

### 5.1 价格锚点设计

- **Free 层存在意义**：不是慈善，是获客引擎。Free 用户的推荐系数约 2.3（每 10 个 Free 用户推荐 23 个新用户）
- **Pro 心理门槛**：$12 是"一杯咖啡钱"，但实际 CAC 通过自助转化 < $5
- **Team 的"老板决策"问题**：$49 × 12 = $588/年，在中小企业的"无需审批"阈值内
- **Enterprise 的价格模糊化**：不公开报价，防止竞品对标，同时允许灵活锚定客户预算
