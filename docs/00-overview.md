## 核心术语

| 术语 | 定义 |
|------|------|
| **Tunnel（隧道）** | 从内网客户端到公网入口的反向代理通道 |
| **Relay（中继）** | OmniTun 云端的流量中转节点，负责将公网请求转发至内网客户端 |
| **Edge（边缘节点）** | 部署在用户就近位置的 Relay 节点，实现全球低延迟接入 |
| **Agent（客户端代理）** | 运行在内网机器上的轻量级守护进程 |
| **Gateway（网关）** | 面向接受端部署的入口网关，可替代 Relay 直连 |
| **Vector（流量向量）** | OmniTun 内部对所有协议的抽象表示，统一 TCP/UDP/HTTP 传输 |
| **Mesh（组网）** | 多个 Agent 之间组建的端到端加密虚拟网络 |
| **Tenant（租户）** | SaaS 多租户体系中的隔离单位，对应一个组织/团队 |
| **Workspace（工作区）** | 租户下的二级隔离域，用于区分环境（生产/测试）或项目 |
